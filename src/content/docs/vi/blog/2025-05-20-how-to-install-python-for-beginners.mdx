Việc cài đặt Python rất dễ dàng. Hãy làm theo các bước dưới đây để cài Python trên máy tính của bạn.

<Tabs>
  <TabItem label="Windows" icon="seti:windows">

<Steps>

1. **Tải trình cài đặt Python**

   Truy cập [trang tải Python chính thức](https://www.python.org/downloads/).  
   Nhấn nút vàng lớn **“Download Python 3.x.x”** (chữ `x.x` là phiên bản mới nhất).

   ![Trang tải Python](/images/blog/2025-05-20-how-to-install-python-for-beginners-python-website-windows.webp)

   Sau đó, mở tệp cài đặt vừa tải về.

2. **Cài đặt Python vào ổ `C:\` mặc định**

   :::tip

   Bỏ qua bước này nếu bạn muốn cài Python vào ổ đĩa khác.

   :::

   Nếu ổ `C:\` của bạn còn thừa dung lượng trống:

   - ✅ Chọn “Add python.exe to PATH”  
   - ✅ Chọn “Use admin privileges when installing py.exe”  
   - Nhấn **“Install Now”**

   ![Cửa sổ cài đặt Python](/images/blog/2025-05-20-how-to-install-python-for-beginners-pythoninstall_firsttime.webp)

   :::note
   Nếu hiện ra hộp thoại này:

   ![UAC trên Windows](/images/blog/2025-05-20-how-to-install-python-for-beginners-windows_uac.webp)

   Chỉ cần nhấn **“Yes”** để tiếp tục.
   :::

3. **Cài đặt Python vào ổ đĩa khác**

   :::tip

   Bỏ qua bước này nếu bạn đã xong bước 2 ở trên.

   :::

   Nếu ổ `C:\` sắp hết dung lượng hoặc bạn muốn cài Python vào nơi khác:

   - Mở **This PC** và tìm ổ đĩa còn trống.

     ![Giao diện This PC](/images/blog/2025-05-20-how-to-install-python-for-beginners-thispc_choosedrive.webp)

   - Trong trình cài đặt Python, chọn **“Customize installation”**

     ![Cửa sổ Tùy chỉnh cài đặt](/images/blog/2025-05-20-how-to-install-python-for-beginners-pythoninstall_customsettings.webp)

   - Tích các ô như trong hình rồi nhấn **“Next”**

   - Trong màn hình kế tiếp:  

     ![Tùy chọn nâng cao](/images/blog/2025-05-20-how-to-install-python-for-beginners-pythoninstall_advancedoptions.webp)

     - Tích các ô như trong hình  
     - Nhấn **Browse** và chọn đường dẫn tùy chỉnh (ví dụ: `E:\python3`) hoặc gõ trực tiếp.  
       _Nên tạo/gõ tên thư mục mới như `python3` để tránh lỗi._

   - Nhấn **“Install”**

   :::note
   Nếu hiện ra hộp thoại này:

   ![UAC trên Windows](/images/blog/2025-05-20-how-to-install-python-for-beginners-windows_uac.webp)

   Chỉ cần nhấn **“Yes”** để tiếp tục.
   :::

4. **Hoàn tất cài đặt**

   Sau khi cài đặt xong:

   ![Cài đặt hoàn tất](/images/blog/2025-05-20-how-to-install-python-for-beginners-pythoninstall_setupcomplete.webp)

   - (Tùy chọn) Nhấn **“Disable path length limit”**
   - Sau đó nhấn **“Close”**

   🎉 Và xin chúc mừng, bạn đã cài xong Python rồi đấy.

</Steps>

</TabItem>
<TabItem label="macOS" icon="apple">

<Steps>

1. **Tải trình cài đặt Python**

   Truy cập [trang tải Python chính thức](https://www.python.org/downloads/).  
   Nhấn nút vàng lớn **“Download Python 3.x.x”**

   ![Trang tải Python](/images/blog/2025-05-20-how-to-install-python-for-beginners-macos-macOS14-2025-05-17-20-31-01.webp)

   Nếu trình duyệt hỏi cái này, nhấn **Allow** để cho phép tải về.

   ![Xác nhận trên macOS](/images/blog/2025-05-20-how-to-install-python-for-beginners-macos-macOS14-2025-05-17-20-31-29.webp)

2. **Mở trình cài đặt**

   Sau khi tải xong, mở tệp `.pkg` để bắt đầu cài đặt.

   ![Cài đặt macOS](/images/blog/2025-05-20-how-to-install-python-for-beginners-macos-macOS14-2025-05-17-20-31-50.webp)

3. **Thực hiện các bước cài đặt**

   - Nhấn **Continue** ở tất cả các bước  

     ![Cửa sổ cài đặt Python](/images/blog/2025-05-20-how-to-install-python-for-beginners-macos-macOS14-2025-05-17-20-32-08.webp)

   - Nhấn "Agree" để đồng ý với điều khoản  

     ![Đồng ý điều khoản](/images/blog/2025-05-20-how-to-install-python-for-beginners-macos-macOS14-2025-05-17-20-32-35.webp)

   - Nhấn **Install**  

     ![Cài đặt Python](/images/blog/2025-05-20-how-to-install-python-for-beginners-macos-macOS14-2025-05-17-20-32-43.webp)

   - Xác thực bằng mật khẩu đăng nhập máy Mac hoặc Touch ID (nếu có hộp thoại xuất hiện)

     ![Xác thực người dùng](/images/blog/2025-05-20-how-to-install-python-for-beginners-macos-macOS14-2025-05-17-20-32-54.webp)

4. **Cài đặt chứng chỉ**

   Sau khi cài đặt xong, cửa sổ Finder sẽ mở ra.

   Nhấn đúp vào **`Install Certificates.command`**

   ![Finder mở chứng chỉ](/images/blog/2025-05-20-how-to-install-python-for-beginners-macos-macOS14-2025-05-17-20-33-27.webp)

   :::tip

   Nếu bạn lỡ đóng cửa sổ Finder hoặc không biết tệp ở đâu, bạn có thể dùng nút Search 🔍 ở góc phải trên cửa sổ Finder để tìm.

   :::

   Chờ Terminal hiện dòng `[Process completed]`, sau đó đóng lại.

   ![Terminal cài chứng chỉ](/images/blog/2025-05-20-how-to-install-python-for-beginners-macos-macOS14-2025-05-17-20-35-34.webp)

5. **Hoàn tất**

   - Nhấn **Close** ở cửa sổ trình cài đặt
   - Chọn **Move to Trash** để xóa tệp cài đặt

   ![Hoàn tất cài đặt Python](/images/blog/2025-05-20-how-to-install-python-for-beginners-macos-macOS14-2025-05-17-20-38-14.webp)

   🎉 Và xin chúc mừng, bạn đã cài xong Python trên máy Mac rồi đấy.

</Steps>
